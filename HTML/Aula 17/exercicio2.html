HTML

        HTML é uma linguagem de marcação utilizada para desenolver sites para a web. 
        É composto por elementos definidos por meio de tags,s
        e suporta estilização e scripts externos como por exeemplo JavaScript.

CSS

        O CSS é a sigla para  Cascade Style Sheet ou folha de estilos em cascata.
        É utilizado para estilizar componentes de uma pagina HTML.
        Pode ser utilizado inline dentro da tag, ou fora dela em uma tag <style></style> ou até mesmo em um arquivo externo.
